# Zkong Converter App

フォルダを常駐監視し、投入された固定長ファイル（汎用機・POSデータ等）やCSVファイルを自動で解析し、外部APIへ高速で転送するオンプレミス向けのデータ変換エンジンです。
全角・半角が混在する日本国内の基幹データに完全適合した、超軽量・安定稼働のデータ連携ソリューションを提供します。

## 🌟 特徴
- **仕様書準拠のバイト単位解析（CP932対応）**
  全角文字を2バイトとして正確にカウントし、仕様書の定義通りに1ミリの狂いもなく切り出します。商品名に漢字が混ざっても、データがズレて文字化けする心配はありません。
- **複数フォーマットの自動判別**
  ファイル名に含まれるキーワード（`match_pattern`）を検知し、複数の固定長/CSV解析ルールを自動で切り替えて処理します。
- **省スペース運用（ファイル自動削除）**
  処理が正常に完了した元ファイルは、ストレージを圧迫しないよう自動的に安全消去されます。
- **大文字・小文字を問わない履歴制御**
  環境変数の設定により、履歴CSVの作成有無を確実にコントロール可能です。

---

## 🚀 実行コマンド

プログラムの起動モードは運用の形に合わせて2種類から選べます。

### 1. 自動監視モード (推奨・リアルタイム運用)
指定されたフォルダ（`data/waiting`）を常駐監視し、新着ファイルが投入された瞬間にリアルタイム検知してAPI転送を行います。
```bash
python converter.py --watch
```

## プロジェクト構成
data内は環境変数の設定で変更可能です。

```text
app/                     <-- コンテナのワークスペース
├── data/                # データ処理スペース
│   ├── active/          # 処理中のファイルが一時的に入る場所
│   ├── history/         # CSVの履歴（設定が true の場合のみ）
│   └── waiting/         # ファイルアップデート先（常駐監視先）
├── format/              # 解析定義フォルダ
│   └── formats.json     # 各フォーマットのルール
├── .env                 # 環境設定ファイル
└── converter.py         # メインプログラム
```
